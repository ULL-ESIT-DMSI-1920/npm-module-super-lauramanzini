 # Installation
```
gh extension install lauramanzini/gh-repo-rename
```

 ## Usage

 ```
 Usage: gh-repo-rename [options]

 Options:
  -V, --version             output the version number
  -r, --repo <reponame>     repo
  -o, --org <organization>  organization
  -n, --name <name>         name
  -h, --help                display help for command
 ```
